## 📌 Introduction

NumPy is a **powerful Python library** for **numerical computing**. It provides efficient support for:

* **Multi-dimensional arrays** 🧮
* **Linear algebra operations** 📐
* **Mathematical functions** 🔢
  This repository contains **hands-on examples** and **exercises** designed to strengthen your NumPy skills.

---

## 📂 Contents of `numPy.py`

This repository includes practical demonstrations covering:

| **Topic**                  | **Description**                                   |
| -------------------------- | ------------------------------------------------- |
| **Array Creation** 🟦      | Create 1D, 2D, and multi-dimensional arrays       |
| **Slicing** ✂️             | Access elements and subarrays efficiently         |
| **Arithmetic Ops** ➕➖      | Perform scalar and array-level operations         |
| **Broadcasting** 📡        | Handle operations between different-shaped arrays |
| **Aggregate Functions** 📊 | Summarize data (mean, min, max, std, etc.)        |
| **Filtering** 🔍           | Select elements based on conditions               |
| **Random Generation** 🎲   | Generate random numbers, shuffle arrays           |

## 🚀 Getting Started

Follow these steps to run the examples:

### **1️⃣ Clone the repository**

```bash
git clone https://github.com/your-username/numpy-examples.git
cd numpy-examples
```

### **2️⃣ Install Dependencies**

```bash
pip install numpy
```

### **3️⃣ Run the Examples**

```bash
python numPy.py
```

> 💡 **Tip:** Uncomment the desired lines of code in `numPy.py` to run specific examples.
